== Description ==

With Karmacracy, you have fun while sharing links. Send cool links, while you improve yourself and get relevant: win awards, get domain clips, talk about revelant words as you get clicks sending good kcies, knowing new people and discovering new content. Check karmacracy and have fun!" />

This wordpress plugin will allow you to insert the widget from karmacracy in all your posts with just a button.

== Installation ==

1. Extract and upload the directory `wp-karmacracy-wdgt` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all

REQUIREMENTS

* Wordpress >= v3.0
